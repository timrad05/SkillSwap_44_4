# SkillSwap

Сервис, где люди обмениваются навыками: один учит, другой учится — и наоборот. Можно смотреть каталог, открывать карточки навыков, регистрироваться и вести профиль.

Командный проект [Яндекс Практикума](https://practicum.yandex.ru/). Репозиторий: [SkillSwap_44_4](https://github.com/timrad05/SkillSwap_44_4).

## Основной функционал

- **Главная** — каталог карточек с фильтрами
- **Регистрация** — пошаговый онбординг из нескольких экранов
- **Профиль** — данные пользователя, навыки, избранное
- **Страница навыка** — подробности, похожие предложения, действия с карточкой
- **Авторизация** — вход для зарегистрированных пользователей

## Моя роль в проекте

- UI-компоненты и их доработка
- Storybook для переиспользуемых компонентов
- Интеграция библиотек: react-datepicker, Swiper

## Архитектура

Проект организован по Feature-Sliced Design:

| Слой | Назначение |
|------|------------|
| `app` | Инициализация, провайдеры, глобальные стили |
| `pages` | Страницы и роуты |
| `widgets` | Крупные блоки интерфейса (карточки, фильтры) |
| `features` | Бизнес-логика (auth, навыки, избранное) |
| `entities` | Модели домена (Skill, User, Request) |
| `shared` | UI-kit, хуки, утилиты |
| `api` | Работа с данными |
